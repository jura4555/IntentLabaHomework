## Question: if (a string with zero)

Will alert be shown?
```javascript
if ("0") {
  alert( 'Hello' );
}
```

### Amswer

Yes, alert will be shown.

## Question: Rewrite 'if' into '?'

Rewrite this if using the conditional operator '?':

```javascript
let result;
if (a + b < 4) {
  result = 'Below';
} else {
  result = 'Over';
}
```

### Amswer

```javascript
let result = (a + b < 4) ? 'Below' : 'Over';
```
